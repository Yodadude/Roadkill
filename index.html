Roadkill

A lightweight .NET 4.0 MVC and NHibernate based wiki engine.

View the Project on GitHub yetanotherchris/Roadkill

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

The official homepage is still Codeplex: roadkill.codeplex.com

Features

  • Light weight, easy to use
  • Supports Creole, Markdown and Mediawiki syntax
  • Search powered by lucene.net
  • Import content and files from Screwturn
  • View previous versions
  • Export straight from screwturn
  • Attachment and file support
  • Tags and tag clouds: no more category soup
  • Full page version history
  • Active Directory support by default
  • Easy to customise and theme
  • Export to XML or .wiki files

Powered by

  • NHibernate - supports SQL Express/Server, mySQL, Sqlite, Postgres
  • ASP.NET MVC 3 / Razor
  • JQuery
